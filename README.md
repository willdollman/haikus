# Haikus

This repository contains haikus about different countries.

## File Naming

- Store each haiku in a text file named `$country.txt` (for example, `iceland.txt` or `japan.txt`).

## Rules

- Each haiku file must be exactly 3 lines long.

## Scripts

- Run `./scripts/test-haikus.sh` to validate line counts.
- Run `./scripts/format-haikus.sh` to remove trailing commas from haiku files.
